# Gestion des Commandes

## Description

Ce projet est une application web développée en PHP PDO et MySQL pour gérer les commandes. Il permet d'ajouter, modifier, supprimer et afficher les commandes enregistrées dans la base de données.

## Technologies utilisées

- PHP
- PDO
- MySQL
- HTML
- CSS
- Bootstrap

## Fichiers du projet

- add_order.php : ajout d'une nouvelle commande.
- edit_order.php : modification d'une commande.
- delete_order.php : suppression d'une commande.
- index.php : affichage de la liste des commandes.
- db.php : connexion à la base de données avec PDO.
- header.php : en-tête du site.
- footer.php : pied de page.
- style.css : mise en forme de l'application.
- gestion_commandes.sql : base de données.

## Fonctionnalités

- Afficher toutes les commandes.
- Ajouter une commande.
- Modifier une commande.
- Supprimer une commande.
- Utiliser PDO pour sécuriser les requêtes SQL.
- Interface simple réalisée avec Bootstrap.

## Installation

1. Copier le dossier du projet dans `htdocs` de XAMPP.
2. Démarrer Apache et MySQL.
3. Créer une base de données nommée `gestion_commandes`.
4. Importer le fichier `gestion_commandes.sql`.
5. Vérifier les informations de connexion dans `db.php`.

## Exécution

Ouvrir le navigateur et accéder à :

```
http://localhost/gestion_commandes/
```
